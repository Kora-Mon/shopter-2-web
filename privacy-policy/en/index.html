Privacy Policy - Shopter

Last updated: April 8, 2026

This Privacy Policy describes how the Shopter app (Android package: com.koramon.shopter2) processes user data.

1. Controller and contact

  • Creator: KoraMon
  • Studio: KoraMon Apps
  • Contact email: koramon.apps@proton.me

2. What data the app processes

2.1 Data entered by the user

The app may store information that the user intentionally enters:

  • wishlist item names,
  • prices, currency, notes,
  • links (URLs),
  • categories and tags,
  • item images (if added by the user).

2.2 Technical and operational app data

The app stores operational data locally on the device as needed for functionality, for example:

  • app settings,
  • history of certain in-app actions,
  • notification and watch/recheck preferences.

2.3 Device permissions

The app may use the following permissions (depending on features used by the user):

  • notifications (POST_NOTIFICATIONS) for price/availability alerts,
  • photos/files (image selection, import/export, and backups),
  • camera/photo gallery where applicable based on the selected image input flow.

3. WebView and web content handling

If the user opens a link in the in-app browser, the app loads the target page to display content and to extract relevant product data (e.g., title/price) when the user triggers this action.

This data is used only for Shopter app features.

4. Cloud backup (optional feature)

Shopter offers optional cloud backup/restore via a Google account.

  • Sign-in is provided via Google Sign-In.
  • When cloud backup is used, data is stored in the user's Google Drive account.
  • This feature is active only when explicitly used by the user.

5. Sharing data with third parties

  • We do not sell user data.
  • We do not share data for advertising profiling.
  • Third-party services may be used only to the extent necessary for app features, especially Google Sign-In and Google Drive API (if the user uses cloud backup) and target websites opened by the user in WebView.

6. Storage and retention period

  • Most data is stored locally on the user's device.
  • Data is retained until deleted by the user or until the app is uninstalled.
  • Cloud backups in Google Drive are retained until the user deletes them from their Google account.

7. Security

We use reasonable technical measures to protect data within the capabilities of a mobile app (e.g., secure storage for sensitive data, system permissions, access restrictions).

8. User rights

The user can:

  • edit or delete data directly in the app,
  • uninstall the app and remove local data,
  • revoke permissions in device settings,
  • delete cloud backups from their Google Drive.

For questions or requests regarding privacy, contact us at: koramon.apps@proton.me.

9. Children

The app is not intended for children under 13 years of age. We do not knowingly collect personal data from children.

10. Changes to this policy

We may update this policy from time to time. The new version will be published at the same URL with an updated date.
